 # 🎯 Project Overview
 
 This repository serves as my Digital Literacy Portfolio, created for my role as a Student Digital Ambassador. The goal of this project is to help peers safely and responsibly navigate the digital world, from building professional online profiles to understanding cybersecurity. It contains all the required deliverables across five modules.
 
 ## 📂 Repository Structure
 
 This repository is organized into the following directories:
 
  • 📑 report/: Contains the full written Project Report covering all tasks.🖼️ task-1-presentation/: Contains the exported Digital Literacy Awareness infographic.
  
  • 🔗 task-2-portfolio/: Contains screenshots of my professional digital profiles and reflection notes.
  
  • ☁️ task-3-platforms/: Contains screenshots of coding platform activity and Google Workspace collaboration tools.
  
  • ✉️ task-4-email-etiquette/: Contains sample professional email drafts and a social media do's and don'ts checklist.
  
  • 🛡️ task-5-cybercrime/: Contains the cybercrime case study writeup and a prevention checklist.
  
  ## 📌 Module Summaries 
  
  ### 🎨 Module 1: Digital Literacy Awareness Infographic
  
  This module focuses on creating a visual resource to help students understand the importance of digital literacy. The folder contains a one-page infographic covering essential topics like safe internet practices and professional online presence.
  ### 🌐 Module 2: Build Your Student Digital Portfolio
  
  A professional digital presence is expected in today's world. This module establishes a foundational online portfolio across platforms like GitHub, LinkedIn, and coding forums to prepare for future academic and professional opportunities.
  ### 🧑‍💻 Module 3: Explore Coding & Collaboration Platforms
  
  This task builds familiarity with essential online tools. It includes participation in a beginner-level coding challenge on platforms like HackerRank or CodeChef, as well as the creation of a Google Form for digital collaboration.

  ### 📧 Module 4: Draft a Professional Email & Etiquette Guide 
  
  Professional online communication is a vital skill. This module demonstrates proper email etiquette through two professional drafts (requesting an extension and applying for an internship) and includes a checklist for responsible social media use.
  ### 🔒 Module 5: Cybercrime Awareness Case Study & Prevention Guide 
  
  Recognizing and avoiding online threats is crucial for students. This module explores a cybercrime scenario and provides an actionable "Stay Safe Online" prevention checklist featuring the National Cyber Crime Portal information
